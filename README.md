# Sentinel-IITB-Hackathon
As Web3 adoption grows, retail investors are increasingly targeted by malicious smart contracts, honeypots, and wallet-draining signatures. Traditional security audits are static and fail to protect users at the point of transaction, where they often sign payload data blindly. "Sentinel" is an Agentic AI-driven security middleware designed to autonomously analyze smart contracts and simulate transactions in real-time.

Operating as a browser extension, Sentinel intercepts pending transactions at the window.ethereum level before the wallet prompt even appears. The backend deploys an autonomous AI agent to immediately fetch the contract’s verified Solidity source code via block explorer APIs. For unverified contracts, the agent utilizes transaction simulation on a forked environment, analyzing the raw bytecode to predict exact state changes—detecting, for example, if a transaction will drain funds without returning expected assets.

The Agentic AI cross-references this behavior against known threat vectors like reentrancy attacks, hidden mint functions, and flagged creator wallets. If malicious logic is detected, the agent proactively blocks the transaction and generates a plain-English risk report. By combining Web3.js, transaction simulation, and Agentic AI, Sentinel bridges the gap between complex cryptographic security and consumer protection, ensuring decentralized finance remains secure.
